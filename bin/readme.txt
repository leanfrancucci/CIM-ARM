Instrucciones para utilizar Autobank en Linux

1. Utilizar el comando copy_install.sh, esto copia el directorio data, initFiles y formatFiles de install
2. Copiar los archivos pk correspondientes, debe haberse licenciado con la MAC correspondiente
3. En caso de que el tariff.exe no detecte correctamente la MAC (ej: no existe eth0) se puede crear un archivo mac.ini con la mac address correspondiente separada por "-" en lugar de ":" y tener mucho cuidado que este archivo con contenga un enter al final
